# Line\_FollowerESP32-S3 based autonomous Line Follower Robot. Programmed in C++ via Arduino IDE, featuring PID control, MPU6050 sensor integration, and wireless debugging via OTA and Telnet.\# 🤖 مشروع روبوت تتبع المسار (Autonomous Line Follower Robot)هذا المستودع يحتوي على المنهجية البرمجية والهندسية لمشروع روبوت تتبع المسار، مصمم ليعمل بكفاءة عالية وسرعة استجابة فائقة.\## 1. العقل المدبر (نوع المتحكم) 🧠\* \*\*المتحكم المعتمد:\*\* لوحة `ESP32-S3 Dev Module` (وتحديداً إصدار YD-ESP32-S3 نسخة N16R8).\* \*\*المواصفات البرمجية:\*\* معالج ثنائي النواة (Dual-core) يعمل بتردد 240MHz، مع ذاكرة فلاش 16MB، وذاكرة عشوائية إضافية (PSRAM) بحجم 8MB.\## 2. بيئة البرمجة واللغة المستخدمة 💻\* \*\*برنامج البرمجة (IDE):\*\* تم استخدام `Arduino IDE` لأدواته المدمجة (Serial Monitor \& Plotter).\* \*\*لغة البرمجة:\*\* `C++` (سي بلس بلس).\## 3. المكتبات البرمجية المستخدمة (Libraries) 📚\* \*\*مكتبات التوازن والحركة (MPU6050):\*\*&#x20; \* `Wire.h`: لبروتوكول I2C.&#x20; \* `Adafruit\_MPU6050.h` و `Adafruit\_Sensor.h`: لقراءات التسارع والدوران.\* \*\*مكتبات البرمجة اللاسلكية (OTA \& Wi-Fi):\*\*&#x20; \* `WiFi.h`, `ESPmDNS.h`, `WiFiUdp.h`, `ArduinoOTA.h`.\* \*\*مكتبة المراقبة اللاسلكية:\*\*&#x20; \* `TelnetStream.h`: لبث قراءات السيريال لاسلكياً.\## 4. الإعدادات الحرجة في Arduino IDE ⚙️لضمان استغلال اللوحة بنسبة 100%، يجب ضبط الإعدادات التالية:\* \*\*Flash Size:\*\* 16MB (128Mb)\* \*\*PSRAM:\*\* OPI PSRAM\* \*\*USB CDC On Boot:\*\* Enabled\* \*\*Partition Scheme:\*\* Minimal SPIFFS (1.9MB APP with OTA) أو 16M Flash (2MB APP/2MB OTA).\## 5. تقنيات المعايرة الاحترافية (OTA \& Telnet) 📡\* \*\*البرمجة عبر الهواء (OTA):\*\* لتعديل الكود (مثل قيم PID) لاسلكياً دون إتلاف منافذ الـ USB.\* \*\*شاشة المراقبة اللاسلكية:\*\* استخدام `PuTTY` أو `Termius` لقراءة الحساسات لحظة بلحظة أثناء عمل الروبوت في الحلبة.
